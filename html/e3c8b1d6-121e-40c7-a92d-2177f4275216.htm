새로운 프로젝트를 생성하려면

  1. 파일 메뉴에서 새로 만들기의 프로젝트를 선택한다.

  2. 프로젝트 형식은 다른언어의 Visual c#을 선택한다.

  3. 템플릿은 Windows Forms 응용 프로그램을 선택하고 확인을 누른다.

  4. 새로운 프로젝트가 만들어졌다면 솔루션 탐색기에서 Form1.cs를 더블클릭하여 디자이너를 연다.

  5. 도구 상자의 일반탭을 열어 앞서 등록한 GridControl을 더블클릭 또는 드래그하여 폼에 새로운 GridControl을 생성한다.
